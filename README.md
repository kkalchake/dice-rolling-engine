## Program Overview
This tool simulates rolling two dice multiple times to analyze probability. It sums two independent integers to generate a bell curve distribution where middle numbers are frequent and edge numbers are rare.

Project Files
dice_simulation.py Contains the configuration, math logic, and user interface

README.md Contains setup instructions and documentation
